OVERVIEW

Send Flowers To Me is a marketing website that was produced to encourage flower lovers to send hints to their significant others, family members and friends. Giving the power to the receiver of the flowers to request in a cute and clever way their preferred choice of a boutique from UrbanStems.

OBJECTIVE

The main goal of this project was for me to become acquainted with utilizing React, Node.js, and JavaScript (ES6). For this project I also managed to do a lot of the coding on the backend to pull data directly from UrbanStems API to showcase the flowers relevant to the zip code the user puts into the website. I also managed to use ES Lint to make sure all of the code was superb to UrbanStems standards.
